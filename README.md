# Game Dev Studio Plugin Marketplace

Plugin marketplace for **Claude Cowork** and **Claude Code**.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **game-dev-studio** | AI-powered game development workflows with 8 specialized agents and 15 slash commands |

## Installation

### Claude Cowork (Desktop App)
1. Cowork 상단의 "+" 클릭
2. "GitHub에서 마켓플레이스 추가" 선택
3. URL 입력: `mishukin101/game-dev-studio-plugin`
4. 플러그인 목록에서 **game-dev-studio** 설치

### Claude Code (CLI)
```bash
claude plugin marketplace add mishukin101/game-dev-studio-plugin
claude plugin install game-dev-studio@game-dev-studio-plugin
```
